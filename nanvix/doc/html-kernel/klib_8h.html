◆ bitmap_first_free()

EXTERN bit_t bitmap_first_free ( uint32_t *  bitmap,
size_t  size 
)

Searches for the first free bit in a bitmap. In order to speedup computation, bits are checked in chunks of 4 bytes.

Parameters
bitmapBitmap to be searched.
sizeSize (in bytes) of the bitmap.
Returns
If a free bit is found, the number of that bit is returned. However, if no free bit is found #BITMAP_FULL is returned instead.
