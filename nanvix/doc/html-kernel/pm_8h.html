◆ sleep()

EXTERN void sleep ( struct process **  chain,
int  priority 
)

Puts the current process to sleep in the chain of processes pointed to by chain, with a priority priority.

If priority if greater than or equal to zero, then the process is set to an interruptible sleeping state. Otherwise, it is put is an uninterruptible sleeping state.

Parameters
chainSleeping chain where the process should be put.
priorityPriority that the process shall assume after waking up.
