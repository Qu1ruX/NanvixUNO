PRIVATE block_t block_alloc ( struct superblock *  sb)

Allocates a disk block by searching in the bitmap of blocks for a free block.

Parameters
sbSuperblock in which the disk block should be allocated.
Returns
Upon successful completion, the block number of the allocated block is returned. Upon failed, #BLOCK_NULL is returned instead.
Note
The superblock must be locked.

Maps the offset off in the file pointed to by ip in a disk block number. If create is not zero and such file by offset is invalid, the file is expanded accordingly to make it valid.

Parameters
ipFile to use
offFile byte offset.
createCreate offset?
Returns
Upon successful completion, the disk block number that is associated with the file byte offset is returned. Upon failure, #BLOCK_NULL is returned instead.
Note
ip must be locked.
