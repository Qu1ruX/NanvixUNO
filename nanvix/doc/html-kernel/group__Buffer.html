◆ bread()

EXTERN buffer_t bread ( dev_t  dev,
block_t  num 
)

Reads the block numbered num synchronously from the device numbered dev.

Parameters
devDevice number.
numBlock number.
Returns
Upon successful completion, a pointer to a buffer holding the requested block is returned. In this case, the block buffer is ensured to be locked. Upon failure, a NULL pointer is returned instead.
Note
The device number should be valid.
The block number should be valid.
