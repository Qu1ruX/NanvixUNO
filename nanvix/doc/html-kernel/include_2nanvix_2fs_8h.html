Maps the offset off in the file pointed to by ip in a disk block number. If create is not zero and such file by offset is invalid, the file is expanded accordingly to make it valid.

Parameters
ipFile to use
offFile byte offset.
createCreate offset?
Returns
Upon successful completion, the disk block number that is associated with the file byte offset is returned. Upon failure, #BLOCK_NULL is returned instead.
Note
